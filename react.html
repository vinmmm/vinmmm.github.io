React Router

Make many pages for your single page app

So, do this first--Import three properties from the React Router object.

import { Router, Route, browserHistory, Link } from 'react-router';

Now do this second--Create a file called New.js, then add the code below to it and import your new file to your App.js file.

class New extends Component {
    render(){ 
        return (
        
            <div>
                <h1>hello new page</h1>
                <Link to="/">App</Link>
            </div>;
        
          )

    }
}

Third thing--Just like you did for New.js, import React router's properties to your App.js file and then add the Link tags with a path to your New.js file as shown below.

<Link to="/new">New</Link>

Fourth thing--Mount React Router and the route components for New.js and App.js.

ReactDOM.render((
  <Router history={browserHistory}>
    <Route path="/" component={App} />
    <Route path="/new" component={New} />
    
    
  </Router>
  ), document.getElementById('root'))

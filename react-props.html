React Props

React provides properties (props) for passing data from parents to children. See differences between state and props here. In the example below we will use JSX and use props in the tag attributes much like we would work with tag attributes in HTML.

So, first, import {Component} from react and create a parent and child component like the code below.

import React, { Component } from 'react';



// Parent Component
class Flintstones extends Component {
render() { 

  return (
<div>
 <ul>
    <ListItem when="1960s" name="Great Gazoo" > 
    <ListItem when="1960s" name="Spacely, Jetsons" > 
    <ListItem when="1980s" name="Scacely, Mudd Club" >
  </ul>
</div>

 );

} 

}
// Child Component
class ListItem extends Component {
render() { 
  return (
<li>
{this.props.when} is {this.props.name}
</li> 

   );

} 

}

export default Flintstones;
